# projec-471-2026

Мета проєкту

Метою цього проєкту є створення консольного бота-організатора подій за допомогою мови програмування Python. Програма дозволяє користувачу додавати, переглядати, редагувати та видаляти події, а також отримувати нагадування про найближчі події.

Бот допомагає організувати особистий розклад і спрощує управління подіями. Для збереження інформації використовується файл формату JSON, що дозволяє зберігати події навіть після завершення роботи програми.

Основні можливості програми:

додавання нових подій;

перегляд списку подій;

пошук подій за датою, періодом або категорією;

перегляд подій на сьогодні, завтра або на тиждень;

редагування та видалення подій;

перевірка конфліктів у розкладі.

Опис логіки та алгоритму роботи бота

Програма працює у консольному режимі та використовує цикл, у якому користувач вводить команди. Після запуску бот виводить привітання та список доступних команд.

Алгоритм роботи програми:

Під час запуску програма завантажує список подій із файлу podii.json.

Виводиться привітання та список доступних команд.

Користувач вводить команду.

Залежно від команди викликається відповідна функція:

додавання події;

перегляд подій;

фільтрація подій;

редагування або видалення події;

перегляд нагадувань.

Під час додавання події програма перевіряє, чи не перетинається вона за часом з уже існуючими подіями.

Якщо конфлікту немає, подія додається до списку та зберігається у JSON-файлі.

Цикл роботи продовжується, доки користувач не введе команду "вийти".

Таким чином, програма дозволяє керувати списком подій у зручному режимі через текстові команди.

Опис структури даних

Для збереження інформації про події використовується структура даних список словників.

Кожна подія представлена словником з такими полями:

id – унікальний ідентифікатор події;

title – назва події;

date – дата події у форматі YYYY-MM-DD;

start_time – час початку події;

end_time – час завершення події;

category – категорія або короткий опис події.

Приклад структури події у файлі JSON:

{
    "id": 1,
    "title": "Лекція з програмування",
    "date": "2026-03-15",
    "start_time": "10:00",
    "end_time": "11:30",
    "category": "Навчання"
}

Усі події зберігаються у файлі podii.json, що дозволяє зберігати дані між запусками програми.

Інструкція зі запуску програми

Для запуску програми необхідно виконати такі кроки:

Встановити мову програмування Python 3.

Створити файл з кодом програми, наприклад bot_events.py.

Скопіювати у файл весь програмний код.

Відкрити командний рядок або термінал.

Перейти до папки з програмою.

Виконати команду:

python bot_events.py

Після запуску програми з’явиться привітання та список доступних команд. Користувач може вводити команди для роботи з подіями.
