
 1. Мета проєкту

Метою даного проєкту є розробка консольного бота-організатора подій за допомогою мови програмування Python. Програма призначена для зручного управління особистими подіями користувача та допомагає організувати розклад.

Основним завданням бота є створення простого інструменту, який дозволяє додавати нові події, переглядати вже створені, редагувати їх або видаляти. Крім того, програма може показувати події на сьогодні, завтра, найближчу подію або події за певний період часу.

Для збереження інформації використовується файл у форматі **JSON**, що дозволяє зберігати всі події між запусками програми. Таким чином, користувач може працювати зі своїм розкладом у будь-який момент.

Основні можливості програми:

* додавання нових подій до розкладу;
* перегляд усіх створених подій;
* перегляд подій за певний період часу;
* фільтрація подій за датою або категорією;
* перегляд подій на сьогодні та завтра;
* визначення найближчої події;
* редагування інформації про події;
* видалення подій зі списку;
* перевірка конфліктів у розкладі.


2. Логіка та алгоритм роботи бота

Програма працює у консольному режимі та взаємодіє з користувачем через текстові команди.

Після запуску програми виконується завантаження збережених подій із файлу podii.json. Якщо файл не існує, створюється порожній список подій.

Далі бот виводить привітання та список доступних команд. Користувач вводить потрібну команду, після чого програма виконує відповідну функцію.

Алгоритм роботи програми складається з таких етапів:

1. Запуск програми.
2. Завантаження списку подій із JSON-файлу.
3. Виведення привітання та списку доступних команд.
4. Очікування введення команди користувачем.
5. Виконання відповідної функції залежно від введеної команди.
6. За необхідності збереження змін у файл.
7. Повторення циклу роботи до введення команди **«вийти»**.

Під час додавання нової події програма виконує перевірку на конфлікти. Якщо нова подія перетинається за часом з іншою подією, бот виводить попередження. Це допомагає уникнути накладання подій у розкладі.

Таким чином, бот дозволяє користувачу ефективно керувати своїм розкладом та швидко знаходити необхідну інформацію про заплановані події.


3. Структура даних

Для зберігання подій у програмі використовується структура даних список словників (list of dictionaries).

Кожна подія описується окремим словником, який містить основні параметри події.

Основні поля події:

* id – унікальний ідентифікатор події;
* title – назва події;
* date – дата проведення події у форматі YYYY-MM-DD;
* start_time – час початку події;
* end_time – час завершення події;
* category – категорія або короткий опис події.

Приклад структури однієї події:

```
{
    "id": 1,
    "title": "Лекція з програмування",
    "date": "2026-03-15",
    "start_time": "10:00",
    "end_time": "11:30",
    "category": "Навчання"
}
```

Усі події зберігаються у файлі **podii.json** у вигляді списку таких об'єктів. Використання формату JSON дозволяє легко зберігати та завантажувати інформацію про події.



4. Інструкція зі запуску програми

Для запуску програми необхідно виконати такі дії:

1. Встановити Python 3 на комп’ютер.
2. Створити файл з програмним кодом, наприклад events_bot.py.
3. Скопіювати у файл весь код програми.
4. Відкрити командний рядок або термінал.
5. Перейти до папки, у якій знаходиться файл програми.
6. Виконати команду запуску:

```
python events_bot.py
```

Після цього програма запуститься, і користувач зможе вводити команди для роботи з подіями.


5. Приклад роботи програми

Після запуску програми користувач бачить привітання та список доступних команд.

```
Вітаю! Я бот-організатор подій.

Доступні команди:
додати подію
показати події
події на тиждень
події сьогодні
події завтра
найближча
фільтр дата
фільтр період
фільтр категорія
редагувати подію
видалити подію
допомога
вийти
```

Приклад додавання нової події:

```
Введіть команду: додати подію

Назва події: Зустріч з командою
Дата (YYYY-MM-DD): 2026-03-15
Час початку (HH:MM): 14:00
Час завершення (HH:MM): 15:00
Категорія: Робота

Подію додано!
```

Перегляд списку подій:

```
Введіть команду: показати події

1. Зустріч з командою | 2026-03-15 14:00-15:00 | Робота
```

Перегляд подій на сьогодні:

```
Введіть команду: події сьогодні
Зустріч з командою 14:00
```

Завершення роботи програми:

```
Введіть команду: вийти
До побачення!
```
